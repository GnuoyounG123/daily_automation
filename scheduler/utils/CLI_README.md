# 浙江大学课表CLI使用说明

## 运行方式

```bash
cd schedule/zju
python3 cli.py
```

## 功能菜单

```
1. 查看课表       - 显示完整课表表格（13节×7天）
2. 查看考试安排   - 显示所有考试时间和地点
3. 导出考试到日历 - 导出iCal文件，可导入日历应用
4. 登录           - SSO登录并缓存会话
5. 清除缓存       - 清除会话或课表缓存
6. 查看缓存统计   - 显示缓存状态
0. 退出
```

## 课表展示

表格格式：
- 行：第1-13节（08:00-21:05）
- 列：周一至周日
- 内容：课程名称 + 地点

时间安排：
```
第1节  08:00-08:45    第8节  15:05-15:50
第2节  08:55-09:40    第9节  16:00-16:45
第3节  09:50-10:35    第10节 16:55-17:40
第4节  10:45-11:30    第11节 18:30-19:15
第5节  11:40-12:25    第12节 19:25-20:10
第6节  13:15-14:00    第13节 20:20-21:05
第7节  14:10-14:55
```

## 缓存机制

- **会话缓存**：登录后保存cookies，有效期24小时
- **课表缓存**：课表数据缓存，减少重复请求
- 首次登录需要输入密码，之后自动使用缓存

## iCal导出

导出的 `.ics` 文件包含：
- 考试名称、时间、地点
- 任课教师信息
- 提前一天提醒

可导入到：
- Apple Calendar
- Google Calendar
- Outlook
- 其他支持iCal格式的应用
