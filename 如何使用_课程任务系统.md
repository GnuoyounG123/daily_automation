# 🎓 课程任务管理系统 - 使用说明

## 新的工作流程

### 每周日 - 复盘与规划

**您在每周日晚上告诉我本周的任务**，例如：

> "这周我要完成数字治理的论文，还有周三有个课堂展示，还要读大数据时代的书。"

**我会帮您**：
1. 理解任务难度和紧急程度
2. 根据您的课程表，合理分配到每天
3. 考虑您的精力状态，避免某天任务过重
4. 直接写入 `weekly_tasks.json` 文件

### 每天早上9:05 - 自动邮件提醒

- 系统读取我已配置好的每日任务
- 发送精美邮件到您的邮箱
- 包含当天课程 + 当天任务 + 温暖关心语

## 配置文件说明

### weekly_tasks.json 结构

```json
{
  "week_of": "2026-04-06",
  "big_tasks_summary": [
    "本周要完成的大任务简述"
  ],
  "daily_tasks": {
    "Monday": [
      {
        "content": "任务内容（支持emoji）",
        "note": "备注说明（可选）"
      }
    ],
    "Tuesday": [],
    "Wednesday": [],
    "Thursday": [],
    "Friday": [],
    "Saturday": [],
    "Sunday": []
  }
}
```

**注意**：
- `week_of` 填写本周一的日期
- `content` 是任务的主要内容
- `note` 是详细说明（会显示在邮件中）
- 某天没有任务就留空数组 `[]`

## 与我对话示例

### 场景1：正常安排

**您**：这周要交公共政策的论文，大概3000字，还有周五有个考试要复习。

**我**：收到！我看看您的课表... 周一周二课比较多，我把论文的大头放在周三周四。周五考试的话，周三周四晚上也留时间复习。周六日任务轻松点。

→ 然后我会更新 weekly_tasks.json

### 场景2：临时调整

**您**：周三突然加了个会议，那天的任务移到周四可以吗？

**我**：没问题，我来调整。周三的任务分散到周四和周五，不会让您太累。

→ 直接修改文件，即时生效

### 场景3：任务完成反馈

**您**：今天任务都做完了！

**我**：太棒了！早点休息吧，明天还有xx任务，但不用急，按节奏来就好。

## 定时任务设置

运行 `setup_all_tasks.bat`（右键管理员运行）设置三个定时任务：

| 时间 | 任务 | 内容 |
|------|------|------|
| 9:00 | 学术简报 | arXiv论文推送 |
| 9:05 | 课程任务 | 当天的课程和任务 |
| 22:00 | 晚间复盘 | 提醒休息 |

---

## 优点

✅ **更灵活** - 我可以根据您的实际情况（课表、精力、临时安排）来分配任务

✅ **更人性化** - 不是机械的"每天平均分配"，而是考虑任务难度和您的状态

✅ **更容易调整** - 临时有事直接告诉我，马上改

✅ **更温暖** - 没有励志鸡汤，只有像朋友一样的关心
