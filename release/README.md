# Daily Automation - 学术自动化助手

## 项目简介

Daily Automation 是一个功能强大的学术自动化助手，帮助您自动爬取学术信息、生成简报、管理日程提醒和课程表。

## 功能特性

- ✅ **多源学术信息爬取**：支持 arXiv、Semantic Scholar、AMiner 等学术数据源
- ✅ **智能关键词筛选**：根据关键词自动筛选相关学术信息
- ✅ **每日简报生成**：自动生成格式化的学术简报
- ✅ **邮件自动发送**：可配置邮件发送功能，定期推送学术简报
- ✅ **日程提醒**：设置定时提醒，不错过重要事项
- ✅ **课程表管理**：管理每周课程安排
- ✅ **双界面支持**：提供 Streamlit 网页界面和 Tkinter 桌面应用

## 快速开始

### 方法一：使用桌面应用（推荐）

1. 双击 `DailyAutomation.exe` 启动配置界面
2. 在界面中配置：
   - 📰 新闻源管理：添加和管理学术信息源
   - 🔑 关键词设置：设置用于筛选学术信息的关键词
   - ⏰ 提醒管理：设置日程提醒时间
   - 📧 邮箱配置：配置邮件发送功能
   - 📅 课程表：管理每周课程安排
3. 点击快速操作区的按钮：
   - 📚 学术简报生成：立即执行学术信息爬取和简报生成
   - ⏰ 每日日程提醒：立即检查并显示日程提醒
4. 查看生成的学术简报：在 `data` 目录中查看生成的 markdown 格式简报

### 方法二：使用命令行

1. 安装 Python 3.8+
2. 运行以下命令：
   - 生成学术简报：`python daily_assistant.py crawl`
   - 检查日程提醒：`python daily_assistant.py remind`
   - 同时执行两者：`python daily_assistant.py all`

## 系统要求

- **桌面应用**：Windows 10/11，无需安装 Python
- **命令行版**：Python 3.8+，需安装依赖

## 定时任务设置

1. 运行 `install.bat` 并选择 "设置定时任务"
2. 或直接运行 `setup_task_scheduler.bat`
3. 系统会自动配置每天定时运行的任务

## 配置说明

### 1. 新闻源配置

在配置界面的 "新闻源管理" 中添加学术信息源，支持以下类型：
- **rss**：RSS 订阅源（如 arXiv）
- **web**：网页抓取（如 Semantic Scholar）
- **json**：JSON API 接口

### 2. 关键词配置

在 "关键词设置" 中添加中英文关键词，系统会根据这些关键词筛选相关学术信息。

### 3. 邮箱配置

在 "邮箱配置" 中设置 SMTP 服务器信息，支持：
- QQ邮箱
- 163邮箱
- Gmail
- Outlook

### 4. 提醒配置

在 "提醒管理" 中设置日程提醒，格式为 HH:MM。

## 常见问题

### Q: 学术简报生成失败怎么办？
A: 检查网络连接，确保新闻源URL可访问，查看 `logs` 目录中的错误日志。

### Q: 邮件发送失败怎么办？
A: 检查邮箱配置是否正确，确保开启了 SMTP 服务，查看 `logs` 目录中的邮件发送日志。

### Q: 定时任务不运行怎么办？
A: 检查 Windows 任务计划程序中的任务状态，确保任务已正确创建。

## 技术支持

如果遇到问题，请查看 `logs` 目录中的日志文件，或联系技术支持。
